主函数：
Hm2_normal: 处理 N = 32, 64, 128
Hm2_sparse:  处理 N = 256, 512

函数用途：
LDL_norm:  一般的LDL^T分解 输入对称正定矩阵A, 输出 A=LDL^T
                   L 存储在 A 的下三角(不包括对角线,L的对角线均为1); D 存储在 A 的对角线.

复现上机报告结果的方法：
1. 运行前修改"矩阵设置"中的"N = xxx", 再单击运行
2. 命令行窗口输出计算用时: (Hm2_sparse 运行时间较长, 会在窗口输出额外等待时间, 故建议在工作区查看结果)
    t1: 一般的 LDL^T 分解   t2: 带状Guass消去法   t3: LDL^T 带状方法
对每一种情况, 重复进行5次取平均值, "理论上"可复现上机报告结果( 由于计算机差异和每次计算的偶然性, 大概率无法完美复现 ）
